⚙ SETTINGS
VIBE.TESTING
AI-POWERED QA PIPELINE · v3
מערכת אוטומטית לניהול מחזור חיי בדיקות מלא — מאפיון ועד דוח סיכום, מונעת על-ידי Claude AI. מתאימה לצוותי QA, מנהלי מוצר ומפתחים.
🔄 שלבי הצינור (Pipeline Phases)
◈ שלב 1 — STP · עץ בדיקות
Test Planning — בניית תוכנית הבדיקות המלאה:
• ניתוח מסמך האפיון שהועלה
• מקורות קלט: מסמך (PDF/DOCX/TXT), URL אתר, נתיב אפליקציה מקומית
• עץ בדיקות 60+ פריטים לפי ISTQB
• מטריצת סיכונים, Traceability Matrix, הערכת מאמץ
• Entry/Exit criteria ו-KPIs
◉ שלב 2 — STD · תסריטי בדיקה
Test Script Design — כתיבת 30+ תסריטים מפורטים:
• כל תסריט כולל: צעדים, נתוני בדיקה, תוצאה צפויה, קריטריוני PASS/FAIL
• חלוקה: functional, negative, integration, security, performance
• מבוסס על STP שנוצר בשלב 1
▶ שלב 3 — RUN · הרצת בדיקות
Test Execution — הרצת הבדיקות:
• מצב AI: סימולציה חכמה עם תוצאות ריאליסטיות
• מצב אמיתי: אוטומציה ממשית עם Playwright (web) או pywinauto (desktop)
• דוח הרצה עם PASS/FAIL/SKIP, שגיאות מפורטות, Root Cause Analysis
• שליחת דוח במייל
◆ שלב 4 — STR · סיכום הרצה
Test Summary Report — דוח מנהלים:
• תקציר לרמת C-Level (250 מילה)
• Quality Scorecard ו-KPIs
• Risk Register, פנקס באגים קריטיים
• המלצה סופית: Go / No-Go לשחרור
• Lessons Learned ו-Automation Roadmap
◈ שלב 5 — דוחות היסטוריה
Reports Dashboard — ארכיון הרצות (כמו Jenkins):
• רשימת כל הרצות האוטומציה שבוצעו
• PASS/FAIL/SKIP לכל הרצה
• פתיחת פרטי כל תסריט שנפל
• ייצוא דוח ל-PDF · שליחה במייל
• דורש הפעלת vibe-runner.py

⚡
CHIPS
Chips צבעוניים לשיפור מהיר: הוספת תסריטים, שיפור עדיפויות, מיקוד בנושא ספציפי.

דשבורד מנהלים

🔑
KEY
עדכון Anthropic API Key. מוצג לאחר ההגדרה הראשונית.
🟢
LOCAL / RUNNER
מצב שרת ה-Runner המקומי. ירוק = vibe-runner.py פועל ומאפשר אוטומציה אמיתית.
🚀 איך מתחילים
מצב AI (ללא התקנה):
1. לחץ ▶ START בחלון הפתיחה והכנס את ה-API Key שלך
2. העלה מסמך אפיון (ATTACH) או הכנס URL בשלב STP
3. לחץ GENERATE בכל שלב לפי הסדר (STP → STD → RUN → STR)

מצב אוטומציה אמיתית (דורש Python):
1. pip install playwright pywinauto anthropic python-dotenv
2. python vibe-runner.py
3. הדות הירוק "RUNNER ✓" יופיע בכותרת
4. בשלב RUN בחר "אוטומציה אמיתית" ובחר את האפליקציה/URL
